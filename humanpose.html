    Retrieve real human body measurements from virtual shape

    Realized in 2022 Tags: Human-pose-estimation, body reconstruction, deep learning, 3D-avatar, neural network

    Presentation:

    This project presents a simple way to generate a realistic virtual 3D avatar from one single 2D image. This avatar maintains the body details like clothes and hair and accurate body shape proportions. In fact, it is shown how it's possible to infer the original body measurements from the virtual ones.

    Developed by researchers at Carnegie Mellon University, can be considered the state-of-the-art approach for real-time human pose estimation. It allows real-time localization of human body parts, such as the shoulders, elbows, and ankles, from an input image or video.

    OpenPose use a bottom up approach, that means that the detection do not start from people detections. Instead, it starts from the keypoints detections. The keypoints are detected using a convolution deep neural network that encode the entire image and produce an embedder. Then, another neural network uses this encoder to produce a Part Affinity Field (PAF) that encodes the spatial relationship (location and orientation ) between the human libs. Then this PAF and the initial gloab embedder are used by another neural network to produce a confidence map for the different body parts. At the end of these steps, the model is able to identify the differents keypoints on the body but it is still necessary to build the Spanning tree to connect all the joint and build the skeleton. This is done by using the Greedy Bipartite graph algorithm that split the problem in a bipartite matching poblem and than matching the adjacent nodes indepenetly also using the infered PAF.

    SMPL is a method that reconstruct 3D human bodies from one or more RGB images by predicting parameters of a statistical 3D body model.

    Now that we are infered general template shape we have to move human mesh accordingly to a new pose. This is called Skinning process.

    Standard skinning methods (based on the linear blend skinning (LBS) method) producess deformation on the final mesh produced.

    SMPL algorithm produces a skinning result by a simple, additive process w.r.t general information learned by data.
